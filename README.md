📌 Project Overview

The Fire Fighter Robot Using Arduino is an embedded systems project designed to detect and extinguish fire automatically. The robot uses flame sensors to identify fire and activates a water pump to suppress it. It also supports Bluetooth-based remote control using ESP32-S and live video streaming through ESP32-CAM, allowing users to monitor and control the robot in real time.

🚀 Features
🔥 Automatic fire detection using flame sensors
💧 Automatic water spraying system
📱 Bluetooth remote control using ESP32-S
📹 Live video streaming using ESP32-CAM
🤖 Manual and Automatic operating modes
⚡ Low-cost and energy-efficient design

⚙️ Working Principle
The flame sensor continuously monitors for fire.
When a flame is detected, the Arduino processes the sensor data.
The robot moves toward the fire source.
The water pump is activated automatically to extinguish the fire.
The user can also control the robot manually through Bluetooth.
ESP32-CAM streams live video for remote monitoring.
