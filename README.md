📌 Project Overview

The Fire Fighter Robot Using Arduino is an embedded systems project designed to detect and extinguish fire automatically. The robot uses flame sensors to identify fire and activates a water pump to suppress it. It also supports Bluetooth-based remote control using ESP32-S and live video streaming through ESP32-CAM, allowing users to monitor and control the robot in real time.

🚀 Features
1.Automatic fire detection using flame sensors
2. Automatic water spraying system
3. Bluetooth remote control using ESP32-S
4. Live video streaming using ESP32-CAM
5. Manual and Automatic operating modes
6. Low-cost and energy-efficient design

🛠️ Components Used
1.Arduino UNO
2.ESP32-S
3.ESP32-CAM
4.Flame Sensor
5.Water Pump
6.Motor Driver (L298N)
7.DC Motors
8.Robot Chassis
9.Battery Pack
10.Jumper Wires

💻 Software Used
1.Arduino IDE
2.C Programming

⚙️ Working Principle
1.The flame sensor continuously monitors for fire.
2.When a flame is detected, the Arduino processes the sensor data.
3.The robot moves toward the fire source.
4.The water pump is activated automatically to extinguish the fire.
5.The user can also control the robot manually through Bluetooth.
6.ESP32-CAM streams live video for remote monitoring.
